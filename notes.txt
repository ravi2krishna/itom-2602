    -> SNMP ?? 

        -> Simple Network Management Protocol 

            -> SNMP is a Protocol(Rules) for "Monitoring" and "Managing"
            Network connected devices (Routers, Switches, Servers, IOT etc) in IP Networks.
            It operates over UDP to track Network performance, detect faults and configure remote devices.

            -> SNMP Port 161 / 162

    -> HTTP/S ?? 

        -> Hyper Text Transfer Protocol (Secured)

            -> For "transmitting web data" over World Wide Web (WWW)

                -> https://www.hdfc.bank.in/

            -> HTTP/S Ports 80 / 443 


    -> SSH ?? 

        -> Secure Shell 

            -> For Encrypted Communication between a Client & Server over Insecure Network 

            -> Remote Logins 

            -> SSH Port 22

    -> Modules / Key Concepts Of ITOM 

        -> CMDB (Configuration Management Database)

            -> The "Backbone" Of ServiceNow.
            All Discovered Assets and their Relationships are 
            stored here as Configuration Items (CI's) 

        -> Configuration Items (CI's) 

            -> Any IT Component that needs to be managed 
            Server, Application, Router, Desktop etc 

        -> Discovery 

            -> The Process of Automatically Identifying and Populating 
            CI's into the CMDB. Uses MID Server(Java Agent) to communicate 
            with the Network(Datacenter).

        -> MID Server (Management, Instrumentation & Discovery)

            -> A Java based Application(Agent) installed Windows / Linux Machine
            within your on-premise or cloud. 

            -> MID Server acts as a Bridge between ServiceNow and target Infrastructure  
        
        
        -> Service Mapping 

            -> Creates a Map linking a Business Service(Ecommerce or Banking) 
            to all of it's dependent Infrastructure Components. 

            
        -> Event Management 

            -> Collects Alerts from External Monitoring Tools(Prometheus, Nagios etc) 
            and correlates them into actionable events.
